Most people involved with any one of the VPAC Member Universities is probably eligible for a VPAC account. To use the VPAC systems you will need a personal account and every personal account must be associated with a VPAC project. While creating a new project is not too hard, its often a better idea to join an existing project if someone you work with already has one. Just follow these steps:

  • All users of VPAC Compute or storage facilities must agree to the VPAC Acceptable Use Policy

  • If you already have an account Login and request accounts/projects under the tasks pane

  • If you don't have a VPAC account:
    • To join an existing project, fill in the Account Form

    • If you cannot become part of an existing VPAC Project, fill in the Project Form
